Simple Ruuvi sensor API
=======================
Simple headless Ruuvi sensor API for saving Ruuvi sensor data to a database. A replacement for Ruuvi Cloud.

Install
-------

`$ npm install`

Use ENV variables as you please:

```
DB_HOST="127.0.0.1"
DB_USER=""
DB_PASSWORD=""
DB_NAME="ruuvi"
SERVER_PORT="8080"
```

Start with `$ npm start`.
